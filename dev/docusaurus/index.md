## How to create multiple docs?

```js
// docusaurus.config.js
{
to: "nixos",
position: "right",
label: "NixOS",
},

// sidebar.js
nixosSidebar: [{ type: "autogenerated", dirName: "nixos" }],

// Create file on `my-website/nixos` instead of `docs/`

```
